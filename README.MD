# Kanbanrs

**Kanbanrs** is simple implementation of kanban in CLI form.

Kanbanrs allows creating multiple user accounts and managing many kanban boards to organize any project.

## Current progress:
### Accounts:
- [x] register
- [x] login

### Boards
- [x] create
- [x] display
- [x] add/edit description
- [x] add/remove user
